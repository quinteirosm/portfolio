Portfolio

Mobile Application for VenoCert: Biometric & NFC Authentication

In collaboration with Global ID SA (EPFL Innovation Park), I designed and developed a hybrid mobile application and its dedicated API for the VenoCert service. This cybersecurity project focuses on high-security document authentication using biometric finger-vein scans, NFC tags, and QR codes.

Technical Stack: Developed the mobile client with Ionic, Angular, and TypeScript, and the backend using Node.js, Express, and Prisma ORM. The infrastructure includes Nginx and Docker for deployment, with full API documentation via Swagger.

Key Deliverables: Managed the complete lifecycle from a functional MVP to a polished prototype ready for demonstrations.

Note on Confidentiality: Due to the sensitive nature of the technology, specific technical details are under a non-disclosure agreement.
